Transportation Analytics Dashboard (Power BI)
📌 Overview

This project is an interactive Power BI dashboard designed to analyze and visualize transportation data. It provides insights into key metrics such as trip performance, distance covered, revenue, and operational efficiency.

The goal of this dashboard is to help stakeholders make data-driven decisions by identifying trends, patterns, and areas for improvement.

📊 Features
📍 Trip Analysis – View total trips, distance, and duration
💰 Revenue Insights – Analyze earnings across routes and time periods
⏱️ Performance Metrics – Monitor efficiency and delays
📅 Time-Based Trends – Daily, monthly, and yearly analysis
🌍 Geographical Insights – Location-based performance tracking
📈 Interactive Filters – Drill down by category, date, or region

Power BI Desktop
Data Modeling & DAX

Data Cleaning & Transformation (Power Query)
